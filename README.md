## How to Use

1. Open `index.html` in your web browser
2. Type a task in the input field and click "Add Task" or press Enter
3. Check the checkbox to mark a task as completed
4. Click "Delete" to remove a task
5. Use the filter buttons to view All, Active, or Completed tasks
6. Click "Clear Completed" to remove all completed tasks at once
